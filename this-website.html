This Website

This website was created as an experiment with Google Antigravity. The project serves as an exploration of how AI agents can be utilized to amplify productiivity and execute a vision without the author needing detailed knowledge

This site is a personal endeavor and is not supported by any oroganization. If you would like to contribute to the site, consider buying me a coffee. While I commonly use Amazon links (especially on the gear pages as an aid to the viewer), I am not an affiliate and do not gain anything from them.

How it was Built

The technical foundation of this site—including the Python-based static site generator, the modular template system, and the automated citation processing—was developed with the support of AI. This allowed for a rapid translation of high-level goals into a fully functional web platform.

Open-source libraries were used to build the site, including:

  • Python - site generation
  • Markdown - content formatting
  • Bootstrap - responsive design
  • StackEdit - markdown editor

AI-Assisted Content Generation

While the primary text and structure were authored by humans, AI played a role in drafting summaries, generating initial code scaffolding, and organizing bibliographic data.

Content & Authorship

While the backend infrastructure and code were assisted by AI, all content, design direction, and aesthetics were provided by the author. This includes:

  • All personal photos and expedition media.
  • Curated lists of field opportunities and student resources.
  • Selection and categorization of specific research publications and references.
  • Detailed course descriptions and pedagogical philosophy.

Any external materials or guides included on this site are appropriately referenced to their original authors and institutions to ensure students have direct access to primary sources.
